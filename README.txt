Krigenheim v0.01 Alpha
Created by Keyan, Justin, and CHATGPT

A fantasy hero management game about recruiting and utilizing heroes of various skills and backgrounds to dominate the known universe. Conquer worlds, level up, and prove ownership of all that was destined to be yours.

Alpha Changelog:

- Added main menu.
- Added options menu.
- Added title art and music.
- Added exit button functionality.
- Added New Game menu draft.

How to Play:

Open main.py and game.ipynb in Jupyter. Then, execute game as code.
